Data Frame (data.frame)

Como vimos antes, una matriz slo puede contener elementos del mismo tipo. Debido a esta limitacin surgen los data frames.

Normalmente un data frame contendr un dataset, con las variables como columnas y las observaciones como filas.

  • Columnas: atributos, variables
  • Filas: observaciones, casos, instancias

Internamente, R maneja los data frames como listas de vectores o factores, todas de la misma longitud.

Podemos asimilarlos a una hoja de clculo tradicional.

Para crear data frames usaremos la funcin data.frame(x, y, ), la cual recibe como parmetros las columnas del dataset.

Creacin de data frames

Es posible crear data frames a mano, pero no es la prctica habitual.

In[2]:
###################################
# Data frames: Creacin           #
###################################
# Creacin de data frame vaco
empty <- data.frame()

# A partir de dos vectores
c1 <- 1:10 # vector de enteros
c2 <- letters[1:10] # vector de strings
df <- data.frame(col1 = c1, col2 = c2)

Operaciones sobre data frames

Anlisis exploratorio de los datos de un data frame:

  • head(df): devuelve las primeras observaciones.
  • tail(df): devuelve las ltimas observaciones.
  • str(df): muestra de forma rpida la estructura de la informacin almacenada.
    • Nmero total de observaciones.
    • Nmero total de variables.
    • Lista con todos los nombres de las variables.
    • El tipo de cada variable.
    • Las primeras observaciones de cada variable.
  • summary(df): muestra los estadsticos bsicos de cada variable.

Manipulacin de data frames

Para unir data frames la mejor manera es usar la funcin rbind(df1, df2, ), obteniendo un data frame resultante con ms filas.

In[9]:
###################################
# Data frames: Manipulacion       #
###################################
# Aadir filas
df <- rbind(mtcars, data.frame(mpg = 22, cyl = 5, disp = 202, hp = 100, drat = 2.56, wt = 3.1, 
                               qsec = 15, vs = 1, am = 0, gear =5, carb = 4, row.names=c("seat")))

Para aadir columnas podemos emplear la funcin cbind(df1, df2, ), pasando como parmetros otro data frame o un vector. Tambin existen otras maneras de aadir columnas.

In[10]:
# Aadir columnas
df$newcolumn <- rep(1, nrow(df))
df[, 'copyofhp'] <- df$hp
df$hp.gear <- df$hp / df$gear
v <- 1:nrow(df)
df <- cbind(df, v)

Indexacin de data frames

Al igual que suceda con las matrices, utilizaremos los corchetes [ ] para indexar data frames. Emplearemos dos nmeros enteros: uno para la fila y otro para la columna [row, column].

Podemos aplicar lo aprendido al indexar matrices para indexar data frames.

Existe una manera rpida de seleccionar una columna, utilizando la expresin df$column df["column"] df[1].

Al indexar podemos obtener vectores o data frames dependiendo como lo hagamos.

  • Al seleccionar filas obtenemos siempre data frames.
  • Al seleccionar mltiples columnas obtenemos siempre data frames.
  • Al seleccionar columnas individuales podemos obtener data frames o vectores.

Unin de data frames

Existe una funcin muy til en R que nos permitir unir dos data frames. La funcin merge recibe los siguientes parmetros:

  • x, y: data frames a combinar.
  • by, by.x, by.y: permiten especificar las columnas por las que se combinarn ambos data frames.
  • all, all.x, all.y: permite seleccionar si queremos obtener todas las filas de ambos data frames (FULL JOIN), todas las del data frame x (LEFT JOIN) o todas las del data frame y (RIGHT JOIN).

Su funcionamiento es similar a los JOIN de SQL.

Informacin sobre data frames

Traps sobre data frames

Al leer data frames desde ficheros normalmente se emplea el argumento stringsAsFactors = FALSE para evitar la coercin a factores.

Se suele evitar emplear nombres en filas y utilizarlos slo en columnas.

No utilizar rbind(df1, df2, ) con factores, ya que algunas veces puede dar problemas (coercin).
