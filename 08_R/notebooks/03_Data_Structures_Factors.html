Factores (factor)

Los factores (factors) son un tipo de estructura de datos para almacenar variables categricas.

Recordatorio:

  • Variable categrica: aquella que puede tomar un nmero limitado de posibles valores (categoras). Ejemplos de variables categricas: gnero de una persona, nacionalidad
    • Variable categrica nominal: aquella que no tiene un orden preestablecido.
    • Variable categrica ordinal: aquella que tiene un orden establecido.
  • Variable continua: aquella que puede tomar un nmero infinito de posibles valores. Ejemplos de variables continuas: peso de una persona, estatura de una persona

A los distintos valores que puede tomar la variable se les denomina niveles, factor levels.

Por qu usar factores?

  • Permiten establecer un orden distinto al alfabtico.
  • Muchos modelos/paquetes de R los emplean

# Unir factores
a <- factor(1:10)
b <- factor(letters[a])
union <- factor(c(as.character(a), as.character(b)))
cross <- interaction(a, b)
# ambos producen un conjunto no-ordenado de factors.
# levels: union: 20; cross: 100
# Items: union: 20; cross: 10

Traps sobre factores

Al leer ficheros con datasets las cadenas de caracteres se convierten automticamente a factores. Al utilizar read.table y read.csv usar el parmetro stringsAsFactors = FALSE.

Si los nmeros de un fichero se factorizan se puede revertir con: as.numeric(levels(f))[as.integer(f)]

Evitar NAs en factores, suelen causar problemas.

La coercin de un objeto a factor suele ser una fuente de problemas.
